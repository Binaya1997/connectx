# Authentication Decisions

## Password Hashing

Selected:
Argon2

Reason:

- Modern password hashing algorithm
- Resistant against GPU attacks
- Better default security than bcrypt
- Recommended for new applications

---

## Authentication Method

Selected:
JWT

Reason:

- Stateless authentication
- Scales horizontally
- Easy mobile integration
- Suitable for microservices architecture

---

## Frontend State Management

Selected:
Zustand

Reason:

- Lightweight
- Simple API
- Less boilerplate than Redux
- Easy authentication state management

---

## Database

Selected:
PostgreSQL

Reason:

- Open source
- ACID compliant
- Excellent JSON support
- Scales well
- Production ready

---

## Frontend Framework

Selected:
Next.js

Reason:

- React based
- Server components
- Excellent performance
- SEO support
- Suitable for large products
